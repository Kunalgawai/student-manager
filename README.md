# Student Manager (Python CLI Project)

This is a beginner Python project to manage student records.

- Add student
- View all students
- Search student
- Delete student
- Data stored in file

## Technologies Used
- Python
- File Handling

## How to Run
1. Clone repo
2. Run: python main.py
